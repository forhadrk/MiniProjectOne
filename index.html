Introduction

Heart disease remains the world’s top cause of death, claiming nearly 18 million lives each year. This group of conditions, including heart attacks and strokes, is often driven by risk factors like high blood pressure, high cholesterol, smoking, and diabetes.

  • Objective

    To analyze key features and build a predictive model for heart disease using Python.

  • Target:

    Predict the presence of heart disease

Dataset Overview

  • → Data Source - Kaggle

    Heart Disease Prediction

  • → Total records

    303

  • → Features

    14, including both categorical and continuous variables

  • → Target Variable

    output (binary variable indicating heart disease presence)
    Feature Description Data Type
    age Patient age integer
    sex Gender (1 = male, 0 = female) integer
    cp Chest pain type (categorical: values 0–3) integer
    rtbpst Resting blood pressure (mm Hg) integer
    chol Cholesterol level (mg/dl) integer
    fbs Fasting blood sugar (>120 mg/dl) (1 = true; 0 = false) integer
    restecg Resting electrocardiographic results (categorical: values 0–2) integer
    thalachh Maximum heart rate achieved integer
    exng Exercise-induced angina (1 = yes; 0 = no) integer
    oldpeak Depression induced by exercise relative to rest float
    slp Slope of the peak exercise ST segment integer
    caa Number of major vessels (0-4) colored by fluoroscopy integer
    thall Thalassemia (categorical: values 0–3) integer
    output Target variable (1 = presence of heart disease, 0 = absence) integer
